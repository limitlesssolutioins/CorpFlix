# LIDUS

**LIDUS** es una plataforma integral de gestión empresarial diseñada bajo cinco pilares fundamentales:

*   **L**iderazgo
*   **I**ntegración
*   **D**esarrollo
*   **U**sabilidad
*   **S**eguridad

## Módulos del Sistema

1.  **Recursos Humanos:** Gestión de empleados, turnos, asistencia y nómina.
2.  **Gestión y Calidad:** Planeación estratégica, riesgos, auditorías y mejora continua.
3.  **Ingeniería Financiera:** Análisis de costos reales, cálculo de precios (pricing) y rentabilidad por proyectos.

## Tecnologías

*   **Frontend:** Next.js (App Router), Tailwind CSS, Lucide React.
*   **Backend:** Node.js, API Routes.
*   **Base de Datos:** SQLite (vía Prisma / Lib DB local) y JSON para configuración rápida.

## Inicio Rápido

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) para ver la aplicación.
